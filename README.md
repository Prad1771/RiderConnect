# 🚀 RiderConnect - Instagram for Bikers

**Social app for 200M+ riders**: Photo feeds, realtime safety maps/SOS, trip tracking, clubs & more (80+ features).

[![FlutterFlow](https://img.shields.io/badge/FlutterFlow-Active-blue)](https://flutterflow.io) [![Firebase](https://img.shields.io/badge/Firebase-Realtime-orange)](https://firebase.google.com)

## 🎯 Features (Phase 1 MVP)
- Infinite-scroll feeds/stories (bike/gear/trip posts)
- Safety map: POIs/EV charging/crash detection/SOS
- Trips: Scheduler/RSVP/live GPX/AI routes
- Profiles/clubs/garage/stats/badges

**Full Spec**: [RiderConnect_Full_Spec.docx](RiderConnect_Full_Spec.docx)

## 🛠 Tech Stack
- **Frontend**: FlutterFlow (visual/AI builder)
- **Backend**: Firebase (auth/DB/storage/realtime)
- **Maps**: Google Maps API
